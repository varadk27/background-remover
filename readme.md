To seamlessly verify the functionality, kindly clone the repository, open it in any IDE and run the code using command given in document. Execute the provided POSTMAN collection using Postman. This will ensure you obtain accurate results.  

Steps to follow:-

1. Clone the repository:
git clone https://github.com/varadk27/background-remover.git

2. Navigate to the project directory:
   cd background-remover

3. Install dependencies:
   pip install -r requirements.txt

4. Run the application:
   uvicorn app.main:app --reload

The API will be accessible at http://127.0.0.1:8000 by default.

Now execute the provided POSTMAN collection using Postman app.
